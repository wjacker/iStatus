# iStatus Logo Concepts

These concepts are tailored for a macOS system monitoring app with a clean, native, real-time feel.

## Preferred Direction

Concept A is the recommended primary direction.

- It combines a rounded app-tile silhouette with a status ring and a live telemetry line.
- It reads clearly at small sizes.
- It feels more like a trusted macOS utility than a security or gaming product.

## Concept A: Status Ring

Message: always-on monitoring, health, activity, precision.

- Rounded square background for app-icon friendliness.
- Open circular ring suggests continuous sampling rather than a closed badge.
- Center telemetry line communicates CPU, memory, network, and device activity.
- Accent dot hints at a live status point.

Best use:

- Primary app icon
- Marketing logo mark
- Splash / onboarding visual

## Concept B: Menu Pulse

Message: menu bar first, glanceable, lightweight, fast.

- Top bar references the macOS menu bar.
- Three compact vertical bars evoke real-time metrics.
- A pulse line underneath adds motion and monitoring meaning.

Best use:

- Website section icon
- Secondary mark
- Small technical badge

## Concept C: Monogram IS

Message: compact brand recognition, more logo-like, less dashboard-like.

- Uses an abstract `i` plus `S` curve inside an app-tile frame.
- More brandable for docs and social surfaces.
- Slightly less explicit about monitoring than Concept A.

Best use:

- Brand lockup
- Social avatar
- Docs header

## Color Direction

- Deep navy: `#0F172A`
- Panel blue: `#12243D`
- Cyan: `#57D0FF`
- Blue: `#3B82F6`
- Mint: `#7EF2C6`
- Slate: `#94A3B8`

## Recommendation

Use Concept A as the primary logo mark, then keep Concept C as a secondary monogram for places where the icon needs to feel more brand-led than metric-led.
